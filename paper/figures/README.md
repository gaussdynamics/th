# Figures Directory

Place all figure files (PDF, PNG, EPS, etc.) in this directory.

## Recommended Formats

- **PDF**: Best for vector graphics (diagrams, plots from LaTeX/TikZ, matplotlib)
- **PNG**: Good for raster images (screenshots, photos)
- **EPS**: Alternative vector format (legacy)

## Naming Conventions

Use descriptive names:
- `architecture_diagram.pdf`
- `calibration_results.pdf`
- `attention_patterns.pdf`
- `route_profile_example.pdf`

## Figure Guidelines

- Use high-resolution images (300+ DPI for print)
- Keep file sizes reasonable (< 5 MB per figure)
- Use consistent styling (fonts, colors, line widths)
- Include captions in LaTeX, not in the figure file itself
