## Adding Citations

1. Add BibTeX entries to `references.bib`
2. Use consistent key naming (e.g., `author_year_topic`)
3. Include all required fields (author, title, journal/booktitle, year, etc.)
4. Cite in text using `\cite{key}` or `\parencite{key}`

## Bibliography Style

The thesis uses `biblatex` with:
- Style: `numeric-comp` (IEEE-like numeric citations)
- Sorting: `none` (appears in citation order)
- Backend: `biber`
